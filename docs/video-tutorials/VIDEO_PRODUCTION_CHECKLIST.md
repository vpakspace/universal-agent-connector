# Video Production Checklist

Complete checklist for producing the 5 video tutorials.

## Pre-Production

### Planning
- [ ] Review all 5 video scripts
- [ ] Prepare demo environment
- [ ] Create sample data
- [ ] Set up test accounts
- [ ] Prepare screen recording software
- [ ] Set up audio recording equipment

### Environment Setup
- [ ] Clean, organized desktop
- [ ] Consistent browser theme
- [ ] Readable font sizes
- [ ] High contrast for visibility
- [ ] Test screen recording quality
- [ ] Verify audio quality

### Content Preparation
- [ ] Sample databases ready
- [ ] Test queries prepared
- [ ] Error scenarios ready
- [ ] Success scenarios ready
- [ ] Screenshots/clips prepared
- [ ] Text overlays designed

## Production

### Recording Setup
- [ ] Screen resolution: 1920x1080
- [ ] Frame rate: 30fps
- [ ] Audio: Clear narration
- [ ] Cursor: Highlighted/visible
- [ ] Zoom: Appropriate for content
- [ ] Background: Clean, professional

### For Each Video

#### Video 1: Setup & Installation
- [ ] Installation steps recorded
- [ ] Dashboard access shown
- [ ] Agent registration demonstrated
- [ ] Connection testing shown
- [ ] Common mistakes addressed
- [ ] Duration: ~5 minutes

#### Video 2: Your First Query
- [ ] Natural language queries shown
- [ ] Query execution demonstrated
- [ ] Results display shown
- [ ] Multiple query patterns shown
- [ ] Common mistakes addressed
- [ ] Duration: ~4 minutes

#### Video 3: Managing Permissions
- [ ] Permission concepts explained
- [ ] Permission granting shown
- [ ] Best practices covered
- [ ] Testing permissions shown
- [ ] Access preview demonstrated
- [ ] Duration: ~6 minutes

#### Video 4: Monitoring & Analytics
- [ ] Query logs shown
- [ ] Cost tracking demonstrated
- [ ] Performance monitoring shown
- [ ] Usage analytics displayed
- [ ] Alerts configuration shown
- [ ] Duration: ~5 minutes

#### Video 5: Troubleshooting
- [ ] Common issues covered
- [ ] Solutions demonstrated
- [ ] Error messages shown
- [ ] Debugging steps shown
- [ ] Help resources provided
- [ ] Duration: ~6 minutes

## Post-Production

### Editing
- [ ] Add intro/outro screens
- [ ] Add chapter markers
- [ ] Add text overlays
- [ ] Highlight important elements
- [ ] Add transitions
- [ ] Sync audio with video
- [ ] Remove mistakes/pauses
- [ ] Add background music (optional)

### Subtitles
- [ ] Generate subtitles
- [ ] Review and correct
- [ ] Time subtitles correctly
- [ ] Export subtitle files
- [ ] Embed in video (optional)

- [ ] Export in MP4 (H.264)
- [ ] Verify quality

- [ ] Test playback
- [ ] Create thumbnails
